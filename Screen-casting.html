Behind the Scenes of My Screencast

Demonstrating how motion animation can be created using Microsoft PowerPoint.

In this project, I focused on demonstrating how motion-based instructional content can be created using simple and accessible tools. My goal was to design an animated learning sequence entirely in Microsoft PowerPoint and then record a narrated screencast explaining each step. I approached the project by breaking down the animation into layers, applying motion paths, and timing transitions to show how static objects can be turned into smooth, dynamic visuals suitable for teaching.

The final output includes a fully animated PowerPoint sequence and a screencast video that walks viewers through the design workflow. The narration highlights how each animation was created, how objects were layered to produce motion effects, and how these ideas can be used by educators to develop visually engaging learning materials.
