Interactive learning through playful engagement and design.

This project highlights my efforts to create an interactive drag and drop learning experience where learners engage with content by performing meaningful actions. The design leverages multimedia learning principles to improve understanding through visual cues, motion, and active participation.

The game aims to make learning intuitive where each drag and drop strengthens comprehension and retention. Learners receive instant feedback, which helps them correct mistakes, stay motivated, and enjoy the process of discovery.

This project demonstrates how playful interaction and thoughtful instructional design can transform abstract learning into tangible understanding.
