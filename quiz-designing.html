Exploring engagement and learning through interactive design.

This project represents my exploration into interactive learning environments where quizzes serve not only as assessments but as tools for engagement and reflection. I designed this quiz experience using principles of game based learning and self-regulated learning to keep learners curious, challenged, and confident.

Each question integrates cognitive scaffolding and adaptive feedback to help learners monitor their understanding while staying motivated. The goal was to make assessment feel like play encouraging growth, not anxiety.

This project demonstrates how thoughtfully designed quizzes can turn evaluation into empowerment, especially for underrepresented learners exploring STEM and technology.
